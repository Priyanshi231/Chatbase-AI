# ChatBase AI

An AI-powered workspace built with the MERN Stack, Gemini API, file uploads, conversation memory, and Retrieval-Augmented Generation (RAG).

## Tech Stack

- React
- TypeScript
- Express
- MongoDB
- Gemini API
- Tailwind CSS
- Shadcn UI
- LangChain (Later)
- ChromaDB (Later)
